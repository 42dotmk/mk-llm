Услугата им овозможува на корисниците да преземаат или купуваат видео игри (вклучувајќи ги и игрите на Xbox Live Arcade и целосните наслови на Xbox One и Xbox 360), додатоци за постоечки игри, демонстрации на игри заедно со други разни содржини како што се слики од гејмер и теми на табла. - Microsoft Windows (или едноставно Windows) е метафамилија на графички оперативни системи развиени, пласирани и продадени од Microsoft. Се состои од неколку семејства на оперативни системи, од кои секоја се грижиме за одреден сектор на компјутерската индустрија со оперативниот систем обично поврзан со IBM PC компатибилен архитектура. Активни семејства на Windows вклучуваат Windows NT, Windows Embedded и Windows Phone; овие може да опфаќаат подфамилии, на пр. Windows Embedded Compact (Windows CE) или Windows Server. Непостоечките семејства на Windows вклучуваат Windows 9x; Windows 10 Mobile е активен производ, неповрзан со непостоечкото семејство Windows Mobile. - На, објавен во февруари 2003 година, е надградена верзија на играта момче напредување Нинтендо е. “СП” во името се залага за “Специјален”.
